# Генераторы записей для психиатрической документации

Этот набор HTML-файлов предназначен для упрощения и автоматизации создания медицинских записей, связанных с психиатрической практикой.  
Каждый файл реализует отдельный генератор, который помогает экономить время и поддерживать стандартизированное оформление документации.

## Psy.html — генератор клишированных фраз

**Psy.html** — это инструмент для автоматизации заполнения психиатрических записей.  
Он позволяет быстро вставлять клишированные и часто используемые фразы. Дополнительно разбит на версии для записей про пациентов-мужчин и женщин: **PsyM.html** и **PsyF.html**.

### Основные функции

- **Текстовое поле** — главное поле для ввода и редактирования текста.  
- **Кнопка «Вставить дату»** — вставляет текущую дату в текстовое поле.  
- **Кнопка «Очистить»** — очищает текстовое поле.  
- **Кнопка «Скопировать всё»** — копирует содержимое текстового поля в буфер обмена.  

### Клишированные фразы

- Фразы разделены по категориям.  
- У каждой фразы есть кнопка **«Вставить»**, которая:
  - копирует саму отдельную фразу в буфер обмена;  
  - вставляет её в текстовое поле (можно добавлять несколько фраз подряд, внести в них правки, а затем скопировать всё сразу).

## PsyMedic.html — генератор записей о лекарствах

**PsyMedic.html** — это инструмент для автоматизации записей, связанных с психофармакологией.  
Он позволяет быстро вставлять названия препаратов, дозировки и обозначения.

### Основные функции

- **Текстовое поле** — используется для формирования записей о назначенных препаратах.  
- **Кнопки вставки препаратов** — добавляют в текстовое поле названия основных медикаментов, они разделены по группам:
- **Депо-инъекции** — отдельные кнопки для вставки названий основных депо-препаратов.  

### Дозировки и обозначения

- **Основные единицы дозировок**:  
  - миллиграммы (мг), миллилитры (мл), капли, проценты (%).  

- **Время и кратность приёма**:  
  - обозначения времени суток (утро, день, вечер, ночь);  
  - количество приёмов в день.  

- **Дробные дозировки**:  
  - вставка фракций (½, ¼, ⅓ и др.) для обозначения долей таблетки.  

- **Динамика дозировок**:  
  - символы-стрелки для указания на увеличение или снижение дозы.  

- **Сокращения и расшифровки**:  
  - все обозначения дублируются сокращённой и полной формой;  
  - удобно как для документации, так и для дачи указаний пациенту при выписке.

## PsyDNL.html — генератор записей о больничных листах

**PsyDNL.html** — это инструмент для автоматизации создания записей об открытии больничных листов.  
Он позволяет указывать даты, номера и условия закрытия листа нетрудоспособности. Генерирует коды больничных, используя код NPVC.

### Основные элементы интерфейса

- **Выбор дат**:  
  - два календаря для указания периода больничного (первый и последний день);  
  - кнопка **«Вставить сегодняшнюю дату»**.  

- **Номера больничных листов**:  
  - отдельное поле для номера листа нетрудоспособности **типа А**;  
  - отдельное поле для номера листа нетрудоспособности **типа В**.  

- **Выбор действия после закрытия**:  
  - выход пациента на работу на следующий день;  
  - продолжение больничного у другого врача.  

- **Кнопка «Генерировать DNL»**:  
  - формирует полный текст с полными номерами и датами больничных листов;  
  - вставляет результат в текстовое поле.  

### Логика генерации

- **Если указаны оба поля (А и В):**  
  - скрипт сам понимает, что лист **типа А** длится **9 дней**;  
  - даты и номера формируются автоматически для обоих листов.  

- **Если указан только номер листа типа В:**  
  - генерируется запись с кодом и датами для листа В на весь период.  

- **Если указан только номер листа типа А:**  
  - генерируется запись с кодом и датами для листа А на весь период. 

- **Если указано следующее: начало 01.10.2025, конец 17.10.2025, номер А 123, номер В 456, после закрытия на работу, то сгенерированная строка будет выглядеть так:**  
  - DNL: 010012202-A-25-123 - 01.10.2025-09.10.2025; 010012202-B-25-456 - 10.10.2025-17.10.2025, darbā 18.10.2025

## PsyDg.html — генератор формулировок диагнозов по МКБ-10

**PsyDg.html** — это инструмент для автоматизации внесения диагнозов в медицинскую документацию.  
Он работает по логике, аналогичной **Psy.html**, но вместо клишированных фраз вставляет полные формулировки диагнозов.

### Основные функции

- **Текстовое поле** — используется для формирования диагностической части записи.

- **Коды по МКБ-10**  
  - рядом с каждым кодом НКБ-10 расположена кнопка вставки;
  - при нажатии в текстовое поле вставляется **полная формулировка диагноза**, соответствующая выбранному коду.

### Назначение инструмента

- ускорение заполнения медицинской документации;
- снижение риска неточностей и разночтений в формулировках диагнозов;
- унификация диагностических записей при работе с психиатрическими и основными соматическими диагнозами.

## TIPCalculator.html — калькулятор TIP-баллов и сроков ресертификации

**TIPCalculator.html** — это вспомогательная утилита для врачей, позволяющая отслеживать статус сертификата и процесс набора TIP-баллов.

### Основные возможности

- **Расчёт срока действия сертификата**  
  Определяет дату окончания текущего сертификата.

- **Дедлайн подачи заявления**  
  Показывает крайний срок подачи документов на продление сертификата.

- **Учёт TIP-баллов**  
  - ввод уже набранных TIP-баллов;
  - отображение текущего прогресса.

- **Сравнение с планом**  
  - расчёт того, сколько TIP-баллов к текущей дате уже должно быть собрано;
  - сравнение фактического количества с плановым.

### Назначение инструмента

- помогает равномерно распределять набор TIP-баллов во времени;
- позволяет понять, укладывается ли текущий темп в план;
- снижает риск необходимости ускоренного и стрессового добора TIP-баллов перед ресертификацией.
